# AI PageOutliner 隐私政策 / Privacy Policy

## 中文版本

### 1. 信息收集与使用

本扩展程序仅在以下范围内处理数据：

- 读取当前打开的AI对话页面内容，用于生成页面大纲
- 在浏览器本地处理页面内容，不会收集或传输任何用户数据
- 不会记录或存储任何对话内容
- 不会追踪用户的浏览行为

### 2. 权限说明

本扩展程序请求的权限用途如下：

- `sidePanel`: 用于显示页面大纲的侧边栏界面
- `activeTab`: 用于访问当前标签页内容
- `scripting`: 用于处理页面内容和生成大纲
- `tabs`: 用于识别支持的网站
- `downloads`: 用于保存导出的Markdown文件到用户本地设备
- `notifications`: 用于通知用户导出过程的状态和提供反馈
- `host_permissions`: 用于识别和访问支持平台上的AI对话内容

### 3. 支持的网站

本扩展程序仅在以下网站运行：

- DeepSeek Chat (`*.deepseek.com`)
- 腾讯元宝AI (`*.yuanbao.tencent.com`)
- ChatGPT (`*.chatgpt.com`)

### 4. 数据安全

- 所有数据处理均在用户浏览器本地完成
- 不会向任何外部服务器发送数据
- 不会保存任何用户数据或对话记录
- 不使用任何分析或追踪工具

### 5. 更新说明

如本隐私政策有任何变更，将通过扩展程序更新时通知用户。

### 6. 联系方式

如您对本隐私政策有任何疑问，请通过[940180276@qq.com]联系我们。

## English Version

### 1. Information Collection and Usage

This extension only processes data within the following scope:

- Reads current AI chat page content to generate page outlines
- Processes page content locally in the browser
- Does not collect or transmit any user data
- Does not record or store any conversation content
- Does not track user browsing behavior

### 2. Permissions Explanation

The permissions requested by this extension are used for:

- `sidePanel`: Displaying the page outline in a side panel interface
- `activeTab`: Accessing current tab content
- `scripting`: Processing page content and generating outlines
- `tabs`: Identifying supported websites
- `downloads`: Saving exported Markdown files to the user's local device
- `notifications`: Informing users about the status of their export process and providing feedback when the export is complete
- `host_permissions`: Identifying and accessing AI chat content on supported platforms (DeepSeek and YuanBao AI) for exporting conversations

### 3. Supported Websites

This extension only operates on:

- DeepSeek Chat (`*.deepseek.com`)
- Tencent Yuanbao AI (`*.yuanbao.tencent.com`)
- ChatGPT (`*.chatgpt.com`)

### 4. Data Security

- All data processing is performed locally in the user's browser
- No data is sent to any external servers
- No user data or chat records are saved
- No analytics or tracking tools are used

### 5. Updates

Any changes to this privacy policy will be communicated through extension updates.

### 6. Contact

If you have any questions about this privacy policy, please contact us at [940180276@qq.com].
